TROUBLESHOOTING
================================================================================

Problem: "No module named 'cv2'"
Solution: Activate venv first! Use batch files or run: venv\Scripts\activate

Problem: "Could not open camera"
Solution: Check camera index in code (0, 1, or 2)

Problem: "No products in database"
Solution: Run: run_register_product.bat

Problem: Slow performance
Solution: Install CUDA PyTorch for GPU acceleration (see above)

Problem: Low accuracy
Solution: Use good lighting and plain background
