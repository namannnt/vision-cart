⚠️ IMPORTANT NOTES
================================================================================

1. ALWAYS use the batch files (they activate venv automatically)
   
   OR manually activate venv:
   cd VisionCart
   venv\Scripts\activate

2. Your packages are in venv/, not system Python!

3. If you see "No module named 'cv2'" → forgot to activate venv!

4. Camera index might need adjustment (0, 1, or 2)

5. GPU not enabled (works but slower) - see RUN_GUIDE.md for GPU setup
